3. 용어¶

3.1. CSS¶

CSS란 화면이나 종이위에 있는 구조화된 문서(HTML, XML)를 어떻게 화면에 보여줄지 기술하는 언어이다.

3.2. 렌더링¶

2D, 3D 모델(scene file)로 부터 디지털 이미지(래스터, 벡터)를 만들어내는 과정을 뜻한다.
